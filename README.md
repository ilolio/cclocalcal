# 1日のスケジュール管理ツール

HTML + JavaScriptで作成した、1日のスケジュールをグラフィカルに管理できるミニツールです。

## 特徴

- **タイムラインクリックで予定作成**: タイムライン上をクリックするだけで、その時刻に予定を作成できます
- **予定クリックで編集**: 既存の予定をクリックすると編集画面が開き、内容を変更できます
- **ドラッグ&ドロップ対応**: スケジュールをドラッグして簡単に時間を変更できます（15分単位で自動調整）
- **モーダルダイアログ**: 美しいアニメーション付きのモーダルで快適な編集体験
- **カスタマイズ可能な表示範囲**: 表示開始時刻と終了時刻を設定できます（例: 9:00〜18:00のみ表示）
- **フルスクリーン対応**: ブラウザの高さ全体を使用し、無駄な空白がありません
- **サーバー不要**: ローカルストレージを使用してデータを保存するため、サーバーが不要です
- **カラフルなデザイン**: 6種類のグラデーションカラーからスケジュールの色を選択できます
- **リアルタイム表示**: 現在の時刻を赤い線で表示します（表示範囲内の場合）
- **統計情報**: 総スケジュール数、総予定時間、空き時間を自動計算して表示します
- **キーボードショートカット**: ESCで閉じる、Enterで保存など便利なショートカット
- **レスポンシブデザイン**: スマートフォンでも快適に使用できます

## 使い方

### 基本操作

1. **ブラウザで開く**
   - `index.html` をブラウザで開くだけで使用できます
   - サーバーのインストールや設定は不要です

2. **表示時間帯を設定**
   - 「表示開始」と「表示終了」で表示する時間帯を設定できます
   - デフォルトは 06:00〜24:00（カスタマイズ可能）
   - 例: 勤務時間のみ表示したい場合は 09:00〜18:00 に設定
   - 設定はブラウザに保存され、次回も維持されます

3. **スケジュールを追加**（3つの方法）

   **方法1: タイムラインをクリック（推奨）**
   - タイムライン上の好きな時刻をクリック
   - モーダルが開き、予定名と詳細を入力
   - 自動的にクリックした時刻が設定されます（デフォルト1時間）

   **方法2: 上部のフォームから**
   - スケジュール名を入力
   - 開始時刻と終了時刻を選択
   - お好みの色を選択
   - 「➕ 追加」ボタンをクリック

   **方法3: キーボード操作**
   - 予定名入力後、Enterキーで保存

4. **スケジュールを編集**
   - 予定ボックスをクリックして編集モーダルを開く
   - 予定名、開始/終了時刻、色を変更
   - 「保存」ボタンで確定、「削除」ボタンで削除

5. **スケジュールを移動**
   - スケジュールボックスをドラッグして、別の時間帯に移動できます
   - 自動的に15分単位に調整されます
   - 表示範囲内でのみ移動可能です

6. **スケジュールを削除**（2つの方法）
   - 予定をクリック → 編集モーダルの「削除」ボタン
   - または「🗑️ 全削除」ボタンですべて削除

### キーボードショートカット

- **Enter**: モーダル内で予定を保存
- **ESC**: モーダルを閉じる
- **クリック外側**: モーダルを閉じる

### データの保存

- スケジュールはブラウザのローカルストレージに自動保存されます
- ページを閉じても、次回開いた時に同じスケジュールが表示されます
- データはブラウザごと、ドメインごとに保存されます

### カラーオプション

6種類のグラデーションカラーから選択できます：
- 紫（デフォルト）
- ピンク
- 青
- 緑
- オレンジ
- ダークブルー

## 技術仕様

- **HTML5**: セマンティックマークアップ
- **CSS3**: グラデーション、フレックスボックス、レスポンシブデザイン
- **JavaScript (Vanilla)**: フレームワーク不要
- **ローカルストレージAPI**: データの永続化

## ブラウザ対応

- Chrome / Edge (推奨)
- Firefox
- Safari
- モダンブラウザであれば動作します
