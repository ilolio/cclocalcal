# 1日のスケジュール管理ツール

HTML + JavaScriptで作成した、1日のスケジュールをグラフィカルに管理できるミニツールです。

## 特徴

- **ドラッグ&ドロップ対応**: スケジュールをドラッグして簡単に時間を変更できます（15分単位で自動調整）
- **カスタマイズ可能な表示範囲**: 表示開始時刻と終了時刻を設定できます（例: 9:00〜18:00のみ表示）
- **コンパクトな表示**: 最大高さ600pxでスクロール可能、無駄な空白を削減
- **サーバー不要**: ローカルストレージを使用してデータを保存するため、サーバーが不要です
- **カラフルなデザイン**: 6種類のグラデーションカラーからスケジュールの色を選択できます
- **リアルタイム表示**: 現在の時刻を赤い線で表示します（表示範囲内の場合）
- **統計情報**: 総スケジュール数、総予定時間、空き時間を自動計算して表示します
- **レスポンシブデザイン**: スマートフォンでも快適に使用できます

## 使い方

### 基本操作

1. **ブラウザで開く**
   - `index.html` をブラウザで開くだけで使用できます
   - サーバーのインストールや設定は不要です

2. **表示時間帯を設定**
   - 「表示開始」と「表示終了」で表示する時間帯を設定できます
   - デフォルトは 06:00〜24:00（カスタマイズ可能）
   - 例: 勤務時間のみ表示したい場合は 09:00〜18:00 に設定
   - 設定はブラウザに保存され、次回も維持されます

3. **スケジュールを追加**
   - スケジュール名を入力
   - 開始時刻と終了時刻を選択
   - お好みの色を選択（デフォルトは紫のグラデーション）
   - 「➕ 追加」ボタンをクリック

4. **スケジュールを移動**
   - スケジュールボックスをドラッグして、別の時間帯に移動できます
   - 自動的に15分単位に調整されます
   - 表示範囲内でのみ移動可能です

5. **スケジュールを削除**
   - 各スケジュールボックス右上の「×」ボタンをクリック
   - または「🗑️ 全削除」ボタンですべて削除

### データの保存

- スケジュールはブラウザのローカルストレージに自動保存されます
- ページを閉じても、次回開いた時に同じスケジュールが表示されます
- データはブラウザごと、ドメインごとに保存されます

### カラーオプション

6種類のグラデーションカラーから選択できます：
- 紫（デフォルト）
- ピンク
- 青
- 緑
- オレンジ
- ダークブルー

## 技術仕様

- **HTML5**: セマンティックマークアップ
- **CSS3**: グラデーション、フレックスボックス、レスポンシブデザイン
- **JavaScript (Vanilla)**: フレームワーク不要
- **ローカルストレージAPI**: データの永続化

## ブラウザ対応

- Chrome / Edge (推奨)
- Firefox
- Safari
- モダンブラウザであれば動作します
